 <!--
            文档流在网页中最底层的一层
                元素主要有两种状态
                    在文档流中
                    不在文档流中(脱离文档流)

                元素在文档流中的特点
                    块元素
                        1.块元素在页面中独占一行,自上向下垂直排列
                        2.默认宽度是父元素的全部
                        3.默认高度是被内容撑开(子元素)

                    行内元素
                        1.行内元素不会独占一行,只占自身大小
                        2.在页面中从左向右水平排列,如果一行之中不能容纳下所有元素,会另起一行,进行从左向右排列
                        3.宽度和高度是被内容撑开的

在HTML中，标签分为：文本级和容器级；
文本级：p、span、a、b、i、u、em
容器级：div、h系列、li、dt、dd
温馨小提示：
(1).容器级的标签，里面可以放置任何东西；文本级的标签里面，只能放置文字、图片、表单元素。
(2).p标签是一个文本级标签。p里面只能放文字、图片、表单元素。其他的一律不能放。
(3).定义列表是一个组标签，不过比较复杂，出现了三个标签：
dl表示definition list 定义列表
dt表示definition title 定义标题
dd表示definition description 定义表述词儿
dt、dd只能在dl里面；dl里面只能有dt、dd

所有的文本级标签都是行内元素，除了p；p是个文本级，但是是个块级元素；
所有的容器级标签都是块级元素

标签设置display:inline;那么这个标签立即变为行内元素


                盒子模型(盒模型,框模型)
                    css将页面中所有的元素都设置为一个矩形盒子
                    将元素设置为盒子后,对页面的布局就变成将不同盒子摆放在不同的位置

                    每个盒子都有以下几个部分组成
                        内容区 content,大小由width,height两个属性来设置
                        内边距 padding
                        边框 border
                        外边距 margin

                边框(border),边框的大小会影响整个盒子的大小
                    三个属性
                        border-width 边框宽度
                            可以指定4个方向的边框宽度
                            1个值 上下左右
                            2个值 上下 左右
                            3个值 上 左右 下
                            4个值 上 右 下 左
                        border-style 边框的样式
                            dotted - 定义点线边框
                            dashed - 定义虚线边框
                            solid - 定义实线边框
                            double - 定义双边框
                            groove - 定义 3D 坡口边框。效果取决于 border-color 值
                            ridge - 定义 3D 脊线边框。效果取决于 border-color 值
                            inset - 定义 3D inset 边框。效果取决于 border-color 值
                            outset - 定义 3D outset 边框。效果取决于 border-color 值
                            none - 定义无边框
                            hidden - 定义隐藏边框
                        border-color 边框的颜色
                            可以通过以下方式设置颜色：

                                name - 指定颜色名，比如 "red"
                                HEX - 指定十六进制值，比如 "#ff0000"
                                RGB - 指定 RGB 值，比如 "rgb(255,0,0)"
                                HSL - 指定 HSL 值，比如 "hsl(0, 100%, 50%)"
                                transparent
                            注释：如果未设置 border-color，则它将继承元素的颜色。

                        border简写属性,通过该属性,可以设置所有边框的相关样式
                        虽然没顺序要求,但是建议按照官方的width,style,color的顺序书写

                        简写属性还可以拆分来写
                            border-top
                            border-right
                            border-bottom
                            border-left

 内边距 padding(填充)
            边框和内容之间的距离就是内边距
            padding-top
            padding-right
            padding-bottom
            padding-left

            内边距也会影响盒子大小 
            背景颜色会延伸到内边距上
            盒子的大小由内容区,内边距,边框共同决定    
            padding 简写,可以指定4个内边框


        外边距 margin
            外边距不影响盒子的大小
            但是会影响盒子的位置
            margin-top;
            margin-right
            margin-bottom
            margin-left

            如果我们设置左和上外边距,会移动元素本身,如果是右和下,会移动其他元素
            margin 简写,可以指定4个方向的距离

        元素的水平方向布局
            元素在父元素水平方向的位置有一下属性共同决定
            margin-left
            border-left
            padding-left
            width
            padding-right
            border-right
            margin-right

            这7个值相加,必须等于父元素的宽度
            如果灯饰不成立,称为过度约束,则灯饰会自动调整
            调整的情况
                如果这7个值,没有auto,则调整margin-right
                有三个值可以设置auto
                width
                margin-left
                margin-right
                    如果某个值设置为auto,则会自动调整auto值,使等式成立

                    如果一个宽度和一个外边距是auto值,则宽度值最大,设为auto的外边距会设为0

                    如果3个都是auto,则外边距是0,宽度最大

                    如果两个外边距auto,宽度固定,则会将外边距设置为相同的值
                    
                    margin:0 auto;水平居中

                    溢出
            如果子元素的大小超过了父元素,则子元素会从父元素中溢出
            使用overflow 属性来设置父元素如果处理溢出的子元素
            可选值
                visible 默认值,子元素会从父元素中溢出,在父元素的外部显示
                hidden 溢出的内容会被剪掉
                scroll 生成2个滚动条,通过滚动条查看完成的内容
                auto 根据需要生成滚动条

            overflow-x 水平方向
            overflow-y 垂直方向

            垂直外边距的重叠现象
                相邻的垂直方向外边距会发生重叠现象
                兄弟元素
                    兄弟之间的相邻垂直外边距取两者之间最大的值(正值)
                特殊情况
                    一正一负,取两者之和
                    两个都是负的,取两者中绝对值最大的
                
                父子元素
                    父子相邻的外边距,子元素的会传递给父元素(上外边距)
                    这个现象我们必须处理
            
            行内元素
                1.行内元素不支持设置宽和高
                2.可以设置padding,但垂直方向不影响布局
                3.可以设置margin,但是垂直方向不影响布局
                4.可以设置border,但是垂直方向不影响布局

            display 用来设置元素显示类型
                可选值
                    block 设置为块元素
                    inline 设置为行内元素
                    inline-block 设置行内块元素
                    table 将元素设置为一个表格
                    none 元素不在页面中显示

            visibility 用来设置元素的显示状态
                可选值
                    visible 默认值 元素在页面中正常显示
                    hidden 元素在页面中隐藏

            display-none 和vibility-hidden 比较,hidden不显示,但是依然占据位置.


            盒子模型的标准
                w3c 标准
                ie 标准


                box-sizing 用来设置盒子模型遵循的标准
                    可选值
                        content-box 默认值 w3c标准
                        border-box ie标准

                box-shadow: 用来设置元素的阴影效果,阴影不会影响页面布局
                第一个值 水平偏移量
                第二个值 垂直偏移量
                第三个值 阴影的模糊半径
                第四个值 阴影的颜色

                浮动,可以使一个元素向父元素的左边or右边移动 
                使用float设置浮动
                    可选值
                     none默认值,元素不浮动
                     left
                     right 

                元素设置浮动后,水平布局的等式便不需要强制成立

                浮动特点
                    1.元素浮动后,完全脱离文档流,不再占用文档流的位置
                    2.元素浮动不会移出父元素
                    3.如果浮动元素的上边是一个没有浮动的块元素(行内元素不可以),则浮动元素无法上移
                    4.如果浮动元素的父元素宽度不够,浮动元素会往下掉,和文字的特点是一个的.
                    5.浮动元素不会超过他上边的浮动兄弟,只能是一样高度
                    6.浮动元素不会盖住文字,(文字环绕图片效果)

                简单总结
                    浮动主要作用就是让页面中块元素可以水平排列

                脱离文档流后的特点
                    块元素
                        1.不独占一行
                        2.宽度和高度默认都是被内容撑开
                    行内元素
                        1.行内元素脱离文档流后会变成块元素

                脱离文档流后,不需要区分行内元素和块元素
